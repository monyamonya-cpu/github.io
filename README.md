# TODO List App (CLI + GUI)
C言語とPythonを使って作成したTODOリストアプリです。  
CLI（C言語）とGUI（Python + Tkinter）両方の操作方法に対応し、共通のCSVファイルでデータを管理しています。

使用技術
CLI: C言語
GUI: Python（Tkinter）
データ保存: CSVファイル


タスクの一覧表示、保存（CLI側）
GUIからの追加・確認（Python側）

補足
短期間での制作を想定し、データの登録、保存といった基本的な機能に絞っています
CLIでCSVに書き込み、GUIで確認できるよう設計
